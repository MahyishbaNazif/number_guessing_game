# 🎮 Number Guessing Game

A console-based number guessing game built in Python.

## Features

- 🎲 Random Number Generation
- 🎯 Three Difficulty Levels
- 📈 High / Low Hints
- 🔢 Attempt Counter
- 🔄 Play Again Option
- ❌ Input Validation

## Technologies Used

- Python 3
- Random Module
- Loops
- Conditional Statements
- Exception Handling

## How to Run

```bash
python number_guessing_game.py
```
